# Tic-Tac-Toe Game

A simple and interactive Tic-Tac-Toe game built using HTML, CSS, and JavaScript.

## Live Demo

https://jatin-jay-31.github.io/Tic-tac-toe-Game/

## Features

* Two-player gameplay
* Turn indicator for each player
* Automatic winner detection
* Draw detection
* Winning cells highlight
* Reset game functionality
* Responsive design

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Challenges Faced

One of the main challenges was implementing the draw condition correctly. The game needed to detect when all nine cells were filled without a winner.

Another challenge was resetting the game state properly, including:

* clearing the board
* resetting the move counter
* removing winner highlights
* restoring the turn indicator

These challenges helped me better understand state management and DOM manipulation in JavaScript.
